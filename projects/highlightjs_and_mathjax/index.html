Ki kare dağılımında k sayısına "degrees of freedom" denir ve bu dağılım $\chi_{k}^{2}$ simgesi ile gösterilir.


Probability density function'ına bakacak olursak da: \[f(x) = \frac{x^{k/2 - 1} e^{-x/2}}{2^{k/2} \Gamma(k/2)} \] 

        public class Main{
            public static void main(String... args){
                double d = 1;
                double degOfFreedom = 12;
                double res = 1;
                res *= Math.pow(d, degOfFreedom/2d - 1);
                res *= Math.exp(-d / 2d);
                res /= Math.pow(2, degOfFreedom/2d);
                res /= gamma(degOfFreedom/2d);
                System.out.println(res);
            }
        }
